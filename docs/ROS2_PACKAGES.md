# ROS 2 packages

| Package | Role |
|---------|------|
| `amr_bringup` | Launch files and robot bringup |
| `amr_nav` | Navigation stack integration |
| `amr_mission` | Mission manager node |
| `amr_bridge` | Web bridge to dashboard |
| `amr_metrics` | Telemetry collection |
| `amr_logger` | Rosbag recording helpers |
| `amr_perception` | Fiducial / perception hooks |
